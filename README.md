# 🚀 CryptoDominicana - Academia de Criptomonedas RD

Bienvenido a la plataforma educativa líder para inversionistas en la República Dominicana (2026).

## 🛠️ Características
- **Calculadora en vivo:** Conversión de DOP a BTC con precios actualizados.
- **Seguridad:** Implementación de Service Workers para una navegación fluida.
- **Guía Legal:** Información basada en las posturas del Banco Central de la RD.

## 💻 Instalación y Uso
Este es un sitio estático. Solo necesitas subir el `index.html` y el `sw.js` a tu hosting para que funcione.

## 🏦 Retiros a Bancos Locales
Nuestra guía explica cómo utilizar el mercado P2P para transferir fondos a **Banreservas** y **Banco Popular** de forma legítima.
